# Chanel test

Test réalisé avec https://vitejs.dev/

Deploy: https://chanel-test.netlify.app/

Node: v18.15.0

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm run dev
```

You can view the development server at `localhost:5173`.

### Production build

```bash
npm run build
```
